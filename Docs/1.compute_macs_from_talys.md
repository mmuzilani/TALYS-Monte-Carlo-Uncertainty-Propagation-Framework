# Module: [1.compute_macs_from_talys.py](https://github.com/user-attachments/files/23739726/1.compute_macs_from_talys.py)

### **Purpose:**  
This script computes **Maxwellian-Averaged Cross Sections (MACS)** from TALYS-generated neutron-capture cross-section data σ(E).  
It converts raw energy-dependent σ(E) values into astrophysically relevant MACS as a function of kT (thermal energy in keV).

In stellar interiors, neutrons are in thermal equilibrium and follow a **Maxwell–Boltzmann** energy distribution.  
Therefore, theoretical cross sections σ(E) must be averaged over this distribution to produce:

### **Maxwellian-Averaged Cross Section (MACS)**  
\[
\text{MACS}(kT) = \frac{2}{\sqrt{\pi}} \frac{1}{(kT)^2}
\int_0^\infty \sigma(E) \, E \, e^{-E/kT} \, dE
\]

This script performs exactly this operation for **four Fe-isotope neutron-capture reactions**:

- ⁵⁶Fe(n,γ) → ⁵⁷Fe  
- ⁵⁷Fe(n,γ) → ⁵⁸Fe  
- ⁵⁸Fe(n,γ) → ⁵⁹Fe  
- ⁵⁹Fe(n,γ) → 60Fe  

The computed MACS values are then exported to a combined output file:

### **Generated Output File:**  
[Fe_MACS_TALYS.csv](https://github.com/user-attachments/files/23739376/Fe_MACS_TALYS.csv)

# What the Script Does

###  1. Loads each TALYS σ(E) file  
- Reads energy in MeV  
- Reads cross section in barn  
- Removes zero/duplicate energies  
- Sorts energy array for smooth integration  

---

###  2. Converts all units to SI  
- MeV → Joule  
- barn → m²  
- kT(keV) → Joule  

---

### 3. Builds a linear interpolator σ(E)  
Because TALYS provides discrete points, the script creates: sigma_interp(E)

so σ(E) becomes continuous for integration.

---

### 4. Computes MACS for multiple kT values  
Using the Maxwellian integral:

$$
\text{MACS}(kT) = \frac{2}{\sqrt{\pi}} \frac{1}{(kT)^2}
\int_0^\infty \sigma(E)\, E\, e^{-E/kT}\, dE
$$


The script prints the MACS for: kT = 5, 10, 15, 20, 25, 30, 40, 50, 60, 80, 100 keV
